Sobre mim

Olá, sou Brenda Nicolle Carlos, desenvolvedora formada em Programação de Sistemas pelo CEDUP JL. Atuo com desenvolvimento web e back-end, criando soluções modernas, funcionais e bem estruturadas.

Tenho experiência com HTML, CSS e JavaScript na construção de interfaces web, além de Python e SQL para sistemas, automações e manipulação de dados. Meu foco é transformar ideias em aplicações reais, organizadas e eficientes.

No desenvolvimento web, trabalho desde o layout até a implementação lógica das funcionalidades, garantindo boa usabilidade, performance e uma experiência fluida para o usuário.

Sou entusiasta de ciência de dados, IA e engenharia de software, áreas nas quais busco evoluir constantemente através de estudos, projetos e novas tecnologias.

Sigo aprimorando minhas habilidades com dedicação, sempre buscando entregar resultados profissionais e soluções inteligentes.
