# Portfólio — Template (fornecido por ChatGPT)

## O que tem aqui
- site estático com páginas: index.html, about.html, skills.html, projects.html, contact.html
- pasta `assets/` com CSS e imagens placeholder
- instruções para personalizar

## Como personalizar
1. Substitua `assets/profile-placeholder.png` pela sua foto (use 1:1).
2. Substitua `assets/project1.png` ... `project4.png` pelos prints dos seus projetos (tamanho mínimo 800x450 recomendado).
3. Atualize textos (nome, e-mail, LinkedIn) diretamente nos arquivos `.html`.
4. Faça deploy: use GitHub Pages, Netlify, Vercel ou outro host estático.
